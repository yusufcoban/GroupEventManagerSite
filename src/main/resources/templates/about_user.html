About Us

What ist SpoGroMa?

SpoGroMa is a eventmanager for employees of Capgemini. Every employee can create his own event and join existings events. Via event details, you have the opportunity to fix a vanue and futher more details.

Functions

  • Create new events
  • Setting events
  • Enter to other events
  • Leave Events
  • Automatic email notification in case of event changes

Destination

SpoGroMa wants to enable employees to organize their own sports activities and to join these created events.Therefore SpoGroMo offers different leisure-time sport groups. These groups can be managed individually by the employees.
